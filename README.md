# Whatsapp-Chat-Analyzer
Whatsapp Chat Analyzer Using pandas- For data manipulation and structuring.

NLTK (Natural Language Toolkit) - For tokenization, stop word removal, stemming, lemmatization.

TextBlob - For basic sentiment analysis and polarity classification.

spaCy - For advanced semantic parsing, Named Entity Recognition (NER), etc.

wordcloud - To visualise frequently used words.

matplotlib - For plotting graphs, pie charts, etc.

seaborn -For attractive, easy-to-use statistical graphics.

scikit-learn - For machine learning models like sentiment classification.

transformers (by Hugging Face) - If using modern transformer-based models for semantic analysis.

urlextract - Extracts URLs from messages in the chat.

emoji - Used to detect, count, or process emojis in messages.

streamlit - Streamlit is an open-source Python library that makes it easy to create and share custom, interactive web apps for data science and machine learning projects.
